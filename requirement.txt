# Pushpak O2 Website Content Requirements

This document outlines all the content and data required from Pushpak O2 to replace mock data with real company information. The website is built with Next.js and uses mock data files in src/lib/ for development. All content needs to be provided in a structured format for seamless integration.

### Hero Section
- Company tagline/subtitle (current: "Engineering the Future of Clean Air Mobility")
- Hero description text
- Hero background video/image (replace /hero-drone-video.gif)
- Call-to-action text and link

### Vision & Mission
- Vision statement
- Extended vision description
- Vision section image (replace /hero-drone-video.gif)

### Core Values
- List of 3 core values with:
  - Title
  - Description
  - Icon name (from Lucide React icons)

### Footer Information
- Social media links (Twitter, LinkedIn, YouTube, GitHub)
- Company contact details:
  - Full company name (current: "Pushpak O2 Aviation Pvt Ltd")
  - Address
  - Email (current: info@pushpako2.com)
  - Phone (current: +91-11-1234-5678)
- Footer links with last updated dates:
  - Privacy Policy URL and date
  - FAQ/Support URL and date
  - Roadmap URL and date

### Statistics
- Aircraft delivered count
- Flight range (km)
- Energy efficiency (%)
- Safety record (%)
